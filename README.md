Comment build un apk :

Requirements : NodeJs, Android Studio, GIT, 

1. `git clone https://github.com/capitainekiwi88-glitch/sae301.git`

2. Dans la racine du projet faire : `npm i` ou `npm install` (identique)

3. `npm run build`

4. `npx cap sync`

5. `npx cap open android`

La dernière commande ouvre android studio. Dans celui-ci il faut aller dans le menu build dans la barre du haut, puis dans 'Genrate APKs' et voilà.
